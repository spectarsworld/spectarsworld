# hybridnand

Independent security researcher focused on smart contract auditing, DeFi security, and blockchain forensics.

The NAND gate is the universal building block of digital logic — any circuit can be built from it alone. I take the same approach to security: reduce complex systems to their fundamental operations, then find where the assumptions break.

## What I Do

- **Smart contract security** — auditing Solidity protocols, finding bugs before they cost money
- **DeFi security research** — oracle manipulation, flash loan attacks, MEV extraction, economic exploits
- **Blockchain forensics** — tracing funds, analyzing exploits post-mortem, understanding attacker methodology
- **Python tooling** — building automation for security analysis workflows

## Stack

Solidity · Python · Foundry · Slither · EVM internals · DeFi protocols · Rust (learning)
